# 🏥 SmartCare AI

SmartCare AI is a Hospital Management System that helps patients and hospital staff manage appointments, doctors, feedback, and healthcare assistance through a simple web-based platform.

Built with React.js, Node.js, Express.js, and MongoDB.

---

## 🚀 Features

### 👤 User Authentication
- User Registration
- User Login
- Secure Route Protection
- Logout Functionality

### 👨‍⚕️ Doctor Management
- View Available Doctors
- Add New Doctors
- Edit Doctor Information

### 📅 Appointment Management
- Book Appointments
- View Appointment History

### 🤖 AI Health Assistant
- Get Doctor Recommendations
- Get Department Suggestions Based on Symptoms

### ⭐ Reviews & Feedback
- Submit Doctor Reviews
- Share Feedback About Services

### 👥 User Management
- View Registered Users

### 🏥 Staff Portal
- Access Administrative Features
- Manage Doctors
- Manage Users
- Manage Appointments

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SuryaSomeswar/SmartCare-AI.git
```

### Install Frontend Dependencies

```bash
cd frontend
npm install
```

### Install Backend Dependencies

```bash
cd ../backend
npm install
```

### Start Backend

```bash
npm start
```

### Start Frontend

```bash
cd ../frontend
npm start
```

# Smart Hospital Management System

## Overview
The Smart Hospital Management System is a web application designed to simplify hospital appointment booking and reduce patient waiting time. Patients can register, log in, view doctors, check available slots, and book appointments online.

## Features
- User Authentication
- Doctor Management
- Appointment Booking
- Appointment History
- Doctor Reviews & Ratings
- Feedback System
- PDF Receipt Download
- Email Notifications
- Symptom-Based Doctor Recommendation
- Admin Dashboard

## Technologies
- React.js
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- Nodemailer
- jsPDF
